oopa38 황윤혁

1. 사각형 버튼, 타원 버튼, 선분 버튼을 이용하여 도형들이 잘 그려지는지 확인
   이때 그리드 체크박스를 눌러 그리드 기능이 작동하는지 도형을 그리며 같이 확인
2. 그려진 도형들이 Ctrl키를 누른 상태로 움직여지는지 확인 
    이때 가장 앞에있는 도형이 먼저 선택 되는지도 같이 확인
3. 그룹화 버튼 클릭 후 도형들을 드래그 하여 도형 그룹화 하기 ( 2번 이상 )
4. 그룹화가 잘 되었는지 그룹화 시킨 도형들을 움직여보기
5. 그룹화된 도형끼리도 그룹화가 잘 되는지 다시 그룹화 시키기
6. 4번을 재시행하여 확인
